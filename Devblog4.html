DevBlog #4 Creating the website

A website for Changelogs, Development Blogs, News, and more!

Red
January 30th 2024

Hey all! If you have read the previous development blogs, you probably wonder why they all have the same release date. The answer is that the website to write the blogs didn't exist yet!

This blog is going to be shorter compared to the others. (I didn't take pictures of developing the website, Sorry!)

Introduction

It all started with wanting to keep a basic changelog to document updates and changes in Brickadia Deathrun. No seriously, I only planned on a document that I would share publicly.

I asked if there was a free way to publicly host/share it without people downloading it directly to their computers/devices.

If I had known I would get hyper-focused on HTML and design a whole website I wouldn't have done it. I ended up way beyond what I originally planned and created a website.

Website

I wanted this website to feel like it was part of Brickadia, so I matched the theme of the official website, I even made a custom logo! You can see it in the header. Its also now the icon of the Discord server

ChangeLog: This page is going to have the official releases of the Deathrun. It's where you can read up on those bug fixes that you want to hear about!

Support Page: The Support page is going to be for The people, and the DR team. Here you can Report players and bugs. Apply for staff, and support the team! It also has our contact information, and who the Team members are.

Blogs (Main): This page is just a home page for the Blogs we write.

Blogs: I wanted to talk more about the development process, and a changelog couldn't do that. So I stole Brickadia's Idea of development blogs. So expect blogs now and then, especially when updates are taking longer than expected

Home: Since we have multiple pages we need this. This page brings together all the pages. Here you can see the 3 most recent Blogs and the 12 most recent updates.

Conclusion

Told you this was going to be short! We have a lot of other stuff to talk about soon! Like Jungle Escape, DeathBot, and a whole slew of other things!

This is the conclusion of the start of the blog series! We are officially caught up with where we are currently in Deathrun Development. These first 4 development blogs have been a pleasure to write, and I hope to see you soon!
