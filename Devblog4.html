DevBlog #4 Creating the website

A website for Changelogs, Development Blogs, News, and more!


Red
April 23rd 2024

Hey all! If you have read the previous development blogs, you probably wonder why they all have the same release date. The answer is that the website to write the blogs didn't exist yet!

This blog is going to be shorter compared to the others. Mainly because I don't think it would be entertaining to share my struggle with HTML.

Introduction

It all started with wanting to keep a basic changelog to document updates and changes in Brickadia Deathrun. No seriously, I only planned on a document that I would share publicly. Ended up making a whole website...

Website

For the website, I aimed to make it seamlessly blend with Brickadia's style, matching the theme of the official website. I even designed a custom logo! You can spot it in the header; it's also the icon for our Discord server.

ChangeLog: This page features the official releases of Deathrun. Here, you can catch up on all those bug fixes and updates.

Support Page: The Support page is there for you—the players—and the DR team. You can report players and bugs, apply for staff positions, and support the team. It also includes our contact information and profiles of team members.

Blogs: I wanted to talk more about the development process, and a changelog couldn't do that. Inspired by Brickadia's development blogs, I'll be sharing updates here, especially when updates are delayed.

Blogs (Main): This page serves as the main hub for our development blogs.

Home: Since we now have multiple pages, this is the central location that brings everything together. Here, you can find the three most recent blogs and the twelve most recent updates.

Conclusion

Told you this was going to be short! Go explore the website and see what everything looks like!

This is the conclusion of the start of the blog series! We are officially caught up with where we are currently in Deathrun Development. These first 4 development blogs have been a pleasure to write, and I look forward to sharing more with you soon.
